-Adds UI textures that were missing from the RR-Engine source.

Simply drag the folder labeled OriginalUIfix into your RR-Engine project.
(I'd recommend under Assets > Simulator > UI)

The OldVEguids folder is for the old Vanilla Enhanced mod that used placeholder guids. It's included for compatibility. Use this to replace the Placeholder folder if your build is based off of Vanilla Enhanced 0.3 or lower. (Found under Assets > Simulator > UI)
